# CyberSecurity Awareness Bot

##  Project Description
This is a C# console-based chatbot designed to educate users about cybersecurity awareness. It interacts with users and provides information on topics such as passwords, phishing, and safe browsing.

##  Features
- Text-based chatbot interaction
- ASCII art header
- User input handling
- Cybersecurity tips and responses
- Input validation
- Console UI enhancements (colors and delays)

##  How to Run
1. Open the project in Visual Studio Code
2. Open the terminal
3. Run the program using:
   dotnet run

##  Project Structure
- Program.cs → Main chatbot logic
- CyberSecurityChatbot.csproj → Project configuration
- GitHub Actions → CI setup (optional)
